# Mines Predictor - Отдельный деплой

## Инструкция по деплою предсказателя

### 1. Подготовка файлов
Убедитесь, что у вас есть следующие файлы:
- `server-predictor.js` - сервер для предсказателя
- `predictor.html` - интерфейс предсказателя
- `predictor-setup.html` - настройка предсказателя
- `vercel-predictor.json` - конфигурация Vercel

### 2. Деплой на Vercel

#### Вариант A: Через Vercel CLI
```bash
# Установите Vercel CLI если еще не установлен
npm i -g vercel

# Войдите в аккаунт
vercel login

# Деплой с отдельной конфигурацией
vercel --prod --name mines-predictor
```

#### Вариант B: Через веб-интерфейс
1. Зайдите на [vercel.com](https://vercel.com)
2. Создайте новый проект
3. Загрузите файлы:
   - `server-predictor.js` (переименуйте в `server.js`)
   - `predictor.html`
   - `predictor-setup.html`
   - `vercel-predictor.json` (переименуйте в `vercel.json`)

### 3. Настройка переменных окружения
В настройках проекта Vercel добавьте:
- `NODE_ENV=production`

### 4. Получение ссылки
После деплоя вы получите ссылку вида:
`https://mines-predictor-xxx.vercel.app`

### 5. Использование
1. Откройте ссылку предсказателя
2. Введите ID игрока из основной игры
3. Предсказатель будет показывать мины и безопасные ячейки

### 6. Связь с основной игрой
Предсказатель подключается к основной игре через API:
- Основная игра: `https://1waion-life.vercel.app`
- Предсказатель: `https://mines-predictor-xxx.vercel.app`

### 7. Тестирование
1. Откройте основную игру
2. Откройте консоль браузера (F12)
3. Скопируйте Player ID из консоли
4. Откройте предсказатель и введите этот ID
5. Начните игру в основной игре
6. Предсказатель должен показать расположение мин

### 8. Возможные проблемы
- Если предсказатель не работает, проверьте CORS настройки
- Убедитесь, что Player ID введен правильно
- Проверьте, что основная игра запущена и активна

### 9. Обновление
Для обновления предсказателя:
1. Внесите изменения в файлы
2. Повторите процесс деплоя
3. Vercel автоматически обновит приложение
